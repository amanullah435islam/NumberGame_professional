# 🎮 Number Guessing Game

> A professional Java console application developed as part of the DecodeLabs Industrial Training Program.

---

# 📖 Project Overview

The Number Guessing Game is a Java console-based application where the system generates a random number and the player attempts to guess it within a limited number of attempts.

The application provides hints such as "Too High" or "Too Low" until the correct number is guessed or all attempts are exhausted.

This project focuses on building programming logic, conditional statements, loops, user interaction, and clean code practices.
